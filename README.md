E-Ticaret Sepeti Konsol Uygulaması

Basit bir **C# konsol uygulaması** ile e-ticaret sepeti programı
Kullanıcılar ürün ekleyip çıkarabilir, sepeti görüntüleyebilir ve toplam fiyatı hesaplayabilir.

Özellikler

- Kullanıcı ve müşteri yönetimi
- Ürün listesi oluşturma
- Sepete ürün ekleme ve çıkarma
- Sepeti listeleme
- Toplam fiyat hesaplama
- Konsol tabanlı etkileşim

Kullanılan Teknolojiler

- C# (.NET 8 )
- Konsol uygulaması
- Nesne yönelimli programlama (OOP)


Kurulum & Çalıştırma

bash
dotnet build
dotnet run yada dotnet watch
